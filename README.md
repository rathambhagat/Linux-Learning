# Linux Learning Public Notes

This repository documents my learning journey through **Introduction to Linux (LFS101)**.
I am using it as a **learning in public** space to capture notes, practice commands, track progress, and build a clean reference I can revisit while I study Linux.

## Repository purpose

- Keep structured notes from the LFS101 course
- Record hands-on practice and command examples
- Organize topics so they are easy to review later
- Show learning progress publicly in a professional format

## Repository structure

```text
Linux-Learning/
├── README.md
├── notes/
│   └── 01-linux-foundations/
│       └── boot-process.md
├── labs/
├── resources/
└── progress/
```

## Folder guide

| Folder | Purpose |
| --- | --- |
| `notes/` | Course notes and topic summaries |
| `labs/` | Hands-on exercises, command practice, and mini tasks |
| `resources/` | Cheatsheets, references, and useful links |
| `progress/` | Learning log, milestones, and reflections |
